# 報告產出與通知規格

## 1. Telegram 報告格式
- **標題**: 🚨 [Ticker] 投資策略分析報告
- **評級**: 強力買入 / 買入 / 持有 / 賣出
- **目標價**: 預測值與安全邊際 (%)
- **一句話摘要**: 核心投資邏輯。
- **按鈕**: [點擊查看完整 HTML 報告]

## 2. HTML Dashboard (Svelte 5)
- **功能**:
    - 展示篩選後的「隱藏價值股」列表。
    - 歷史分析報告的存檔。
    - 自動渲染「Football Field Chart」(估值對比圖)。
- **自動化**:
    - 分析完畢後，由 Python 腳本產生一個靜態 JSON 檔案供 Svelte 讀取。
    - 透過 Vercel Cron Jobs 每週自動執行全市場篩選。
